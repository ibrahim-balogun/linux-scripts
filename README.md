# Linux Shell Scripting Projects

## Overview
Collection of Linux shell scripts and configuration files
created while learning Cloud DevOps engineering.

## Scripts Included
- myscript.sh - Basic shell script with system info
- servercheck.sh - Server health check script
- createfolders.sh - Automate folder creation with loops
- looptest.sh - Demonstrate bash loops
- iftest.sh - Demonstrate if statements
- variables.sh - Demonstrate bash variables

## Configuration Files
- person.yml - YAML configuration example
- servers.yml - Server configuration in YAML
- person.json - JSON configuration example
- servers.json - Server configuration in JSON
- deployment.yaml - Kubernetes deployment file
- service.yaml - Kubernetes service file

## Skills Demonstrated
- Bash scripting
- Linux administration
- YAML and JSON
- Kubernetes configuration
